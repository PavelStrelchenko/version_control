# Подсказка по гиту

## 1. Инициализация репозитория

```sh
git init
```

## 2. Статус репозитория

```sh
git status
```

## 3. Работа с удалёнными репоиториями

- Сменить имя главной ветки:

```sh
git branch -M <banchname>
```

- Добавить новую ветку и перейти на неё:

```sh
git checkout -b <branchname>
```

- Подключить удалённый репозиторий

```sh
git remote add origin <link>
```

- Отображение привязанного удалённого репозитория:

```sh
git remote show
```

- Вытолкнуть на сой удалённый репозиторий:

```sh
git push --set -upstream origin <repname>
```

- Вытянуть информацию с удалённого репозитория:

```sh
git pull
```
